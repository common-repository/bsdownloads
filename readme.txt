Wordpress plugin that allows users to easily list all attachments associated with a specific post or page. It also allows to list files located in specific directory on file system of wordpress hosting server (files can be uploaded via FTP).

== Description ==

Plugin provides shorttag [bsdownloads] to insert list of file download links into generated HTML content.

== Installation ==

1. Install and activate the plugin through the 'Plugins' menu in WordPress
2. Use shorttags [bsdownloads] to insert list of downloads into post content
3. That's it! :)
